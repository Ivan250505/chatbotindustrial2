Quiero que actúes como un arquitecto de software senior especializado en desarrollo backend con Node.js, integración con WhatsApp Cloud API y bases de datos MySQL en la nube.

Necesito que me desarrolles la arquitectura y el código base para un chatbot oficial de WhatsApp como parte de la convocatoria de la Coordinación del Programa de Ingeniería Industrial de las UTS: “Programa de Jóvenes Consultores Empresariales”.

Contexto del proyecto

El chatbot será usado para gestionar la inscripción y recopilación de información de los participantes de la convocatoria.

El chatbot debe cumplir las siguientes condiciones:

El usuario inicia la conversación mediante un link directo a WhatsApp (Click to Chat).

El objetivo es aprovechar la ventana de conversación de 24 horas de WhatsApp Cloud API, donde los mensajes no generan costo si el usuario inicia la conversación.

Una vez el usuario inicia el chat, el bot debe:

Dar la bienvenida.

Explicar brevemente el programa.

Guiar paso a paso al usuario mediante preguntas estructuradas.

Guardar todas las respuestas en una base de datos MySQL en la nube.

Manejar estados conversacionales por usuario (flujo paso a paso).

Requisitos técnicos obligatorios

Desarrollar usando:

Node.js

Express

MySQL (en la nube)

WhatsApp Cloud API (Meta oficial)

Variables de entorno con dotenv

Arquitectura limpia (modular, no todo en un solo archivo)

Seguridad y configuración

El sistema debe solicitar y usar variables de entorno para:

WHATSAPP_TOKEN
PHONE_NUMBER_ID
VERIFY_TOKEN

DB_HOST
DB_USER
DB_PASSWORD
DB_NAME

No deben quedar credenciales expuestas en el código.
